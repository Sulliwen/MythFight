# Game Workspace

Ce dossier contient le code et les donnees du jeu MythFight.

## Structure
- `src/`: code source gameplay/systems
- `data/`: definitions de contenu (factions, unites, equilibrage)
- `tests/`: tests techniques et tests de logique gameplay
