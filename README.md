# MythFight

Projet de jeu type Castle Fight base sur des factions mythologiques.

## Structure
- `docs/`: vision, game design, economie, technique, production, tests.
- `game/`: code source, donnees gameplay, tests techniques.
- `assets/`: references visuelles et concepts.
- `tools/` et `scripts/`: outillage et automatisation.
