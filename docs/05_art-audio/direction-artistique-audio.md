# Direction artistique + audio

## Placeholders integres

Les placeholders actuels ajoutes pour `soldier`, `griffon`, `Caserne` et `Perchoir` sont des assets temporaires locaux stockes dans `apps/web/public/sprites/placeholders/`.

### Provenance

- `soldier`
  - Source: Openclipart, "Soldier"
  - URL: https://openclipart.org/detail/293426/soldier
  - Usage local: rasterise/retouche minimale pour lisibilite in-game
- `griffon`
  - Source: Openclipart, "Griffin"
  - URL: https://openclipart.org/detail/340849/griffin
  - Usage local: rasterise/retouche minimale pour lisibilite in-game
- `Caserne` et `Perchoir`
  - Source: OpenGameArt, "SVG Isometric Buildings"
  - URL: https://opengameart.org/content/svg-isometric-buildings
  - Fichier source utilise: `BuildingsSpriteSheet.png`
  - Usage local: crops de deux batiments distincts pour placeholders de recrutement

### Licence

- Openclipart: domaine public / usage libre selon la page source.
- OpenGameArt `SVG Isometric Buildings`: CC0 selon la page source.

### Note

Ces assets ne fixent pas la direction artistique finale. Ils servent uniquement a debloquer l'integration gameplay/UI et pourront etre remplaces par des sprites de faction coherents plus tard.
